Mathematische Funktion:

Wenn die Einheit g ist wird das Ergebnis mit 1000 multipliziert,
wenn sie t wird das Ergebnis mit 1000 dividiert
und wenn sie kg ist bleibt es gleich.

Funktion in Javascript:


    const trommel = 1;
    const kabel = 0.12;

    function Gewicht_berechnen(meter, einheit){
        const ergebnis = meter * kabel + trommel;
        if(einheit == "kg"){
            return ergebnis;
        } else if(einheit == "t") {
            return ergebnis / 1000;
        } else {
            return ergebnis * 1000;
        }
    }
