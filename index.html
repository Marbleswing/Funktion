Szenario:

Eine Kabelrolle wiegt ohne Kabel 1 kg.
Ein Meter Kabel wiegt 0,12 kg.

Mathematische Funktion:

f(x) = 0,12x + 1

• x= Die Länge des Kabels in Metern

• 0,12x= Das Gewicht des Kabels

• 1= Das Gewicht der leeren Kabelrolle

Graph:

Funktion in Javascript:


    const trommel = 1;
    const kabel = 0.12;
    
    function Gewicht_berechnen(meter, einheit){
        const ergebnis = meter * kabel + trommel;
        if(einheit == "G" || einheit == "g"){
            return ergebnis * 1000;
        } else if(einheit == "T" || einheit == "t") {
            return ergebnis / 1000;
        } else {
            return ergebnis;
        }
    }

Erklärung:


    function Gewicht_berechnen(meter, einheit)

So definiert man die Funktion in Javascript, meter und einheit sind die Werte die man übergibt. 


    const trommel = 1;
    const kabel = 0.12;

Mit "const" erstellt man eine sogenannte "Konstante".
In dem Fall das Gewicht der Trommel alleine und das Gewicht vom Kabel pro Meter. 


    const ergebnis = meter * kabel + trommel;

In dem Abschnitt wird die Funktion angewendet.
Zusätzlich wird auch in die angegebene Einheit umgerechnet: 


    if(einheit == "G" || einheit == "g"){
        return ergebnis * 1000;
    }

Mit "if(einheit == "G" || einheit == "g")" wird überprüft ob die Einheit G oder g ist.
Wenn das zu trifft wird mit "return ergebnis * 1000;" das Ergebnis in gramm um gerechnet und zurück gegeben.
Das wird dann noch mit den anderen Einheiten gemacht. 


    Gewicht_berechnen(50, g);

So wird die Funktion verwendet.
Mit dem Beispiel wird die Länge 50 übergeben und die Einheit g.
